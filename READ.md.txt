# 🍔 Online Food Delivery Analysis Dashboard

## 📌 Project Overview

This project analyzes online food delivery data to uncover insights related to customer behavior, restaurant performance, payment preferences, delivery efficiency, and revenue trends. The analysis is performed using Python, SQL, Power BI, and Streamlit.

---

## 🎯 Objectives

* Analyze food delivery orders and revenue trends.
* Identify top-performing restaurants.
* Understand customer payment preferences.
* Analyze order distribution across cities.
* Build an interactive dashboard for business insights.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQL
* Power BI
* Streamlit
* VS Code
